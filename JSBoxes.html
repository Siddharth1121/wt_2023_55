Types of Boxes in JavaScript

div

The <div> element is a generic container for flow content. It is often used for grouping and styling purposes.

alert

The alert() function displays a simple dialog box with an alert message and an "OK" button.

prompt

The prompt() function displays a dialog box with a message, an input field, and "OK" and "Cancel" buttons.

confirm

The confirm() function displays a dialog box with a message and "OK" and "Cancel" buttons, allowing the user to make a binary choice.
